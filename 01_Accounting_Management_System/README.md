# Accounting Management System

## Overview

This project is a Microsoft Excel-based Accounting Management System designed to manage accounting transactions and generate financial reports.

## Features

- Chart of Accounts
- Journal Entries
- General Ledger
- Trial Balance
- Income Statement
- Balance Sheet
- Dashboard

## Excel Skills

- VLOOKUP
- SUMIFS
- IF
- Pivot Tables
- Pivot Charts
- Data Validation
- Dashboard Design

## Workbook Structure

- README
- Dashboard
- Summary
- Chart of Accounts
- Journal Entry
- General Ledger
- Trial Balance
- Financial Statements

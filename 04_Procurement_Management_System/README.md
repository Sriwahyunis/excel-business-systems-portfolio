Procurement Management System										
Company										
PT Nusantara Office Supplies										
										
Objective										
This project is a Microsoft Excel-based Procurement Management System designed to manage the procurement process from Purchase Requisition (PR) to Purchase Order (PO), Goods Receipt (GR), procurement reporting, and an interactive dashboard.										
										
										
										
Features										
✅ Department Master										
✅ Supplier Master										
✅ Category Master										
✅ Item Master										
✅ Purchase Requisition										
✅ Purchase Order										
✅ Goods Receipt										
✅ Procurement Report										
✅ Dashboard										
										
Excel Skills										
-	SUMIFS									
-	IF									
-	TODAY									
-	VLOOKUP/XLOOKUP									
-	Data Validation									
-	Conditional Formatting									
-	Charts									
-	COUNTIF									
-	COUNTA									
-	Pivot Tables

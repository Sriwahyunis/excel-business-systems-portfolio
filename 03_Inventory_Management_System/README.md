Inventory Management System										
Company										
PT Nusantara Office Supplies										
										
Objective										
This project is a Microsoft Excel-based Inventory Management System developed to record stock movements, monitor inventory levels, calculate inventory value, identify low stock items, and present key inventory insights through an interactive dashboard.										
										
										
										
Features										
✅ Master Data										
✅ Stock In										
✅ Stock Out										
✅ Inventory Report										
✅ Stock Adjustment										
✅ Dashboard										
										
Excel Skills										
-	SUMIFS									
-	IF									
-	Data Validation									
-	VLOOKUP/XLOOKUP									
-	Pivot Tables									
-	Conditional Formatting									
-	Charts
